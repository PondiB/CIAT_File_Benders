ABOUT
~~~~~~

The File Extractor recusively goes through subdirectories of the root folder identifying
.zip files and extracting them to a destination folder in a single run. For this script
to work, you must expplicitly set your source and destination directory in the
accompanying "directory_paths.json" file.

This script is intended for general use and no warranty is implied for
suitability to any given task. Please feel free to copy/paste/edit to suit your
needs and remember to share ;)
